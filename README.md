A high-performance, engineering-first portfolio showcase built with **Next.js 16 (App Router)** and **TypeScript**. Optimized for the Vercel edge runtime, this project features dynamic GitHub integration, real-time production snapshots, and refined professional aesthetics.

---

## 🚀 Architectural Core

- **Framework**: Next.js 16.1.1 (App Router) with React 19.
- **Styling**: Tailored Dark-Mode UI with custom Green Architecture (`#33a381` to `#129840`).
- **Animations**: Advanced motion effects via **GSAP 3.12** and **WOW.js**.
- **Authentication**: Secure user management via **Clerk**.
- **Data Engine**: Real-time project fetching from the **GitHub REST API**.

## 🛠️ Local Environment Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/professionaldev527/NextJS-theprofdev.com.git
   ```

2. **Configure Environment Variables**:
   Create a `.env.local` file with the following keys:
   - `VERCEL_API_KEY`: For deployment synchronization.
   - `RESEND_API_KEY`: For enabling the contact form mailbox.
   - `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, etc. (for media logic).

3. **Install & Run**:
   ```bash
   npm install
   npm run dev
   ```
